Установка

iOS: 
1. При наличии JailBreak (iOS Superuser) установить ipa с помощью любого доступного инсталятора
2. При отсутствии JailBreak требуется создать аккаунт разработчика Apple, создать новое приложение на сайте appstoreconnect, 
затем загружаем ipa через веб браузер на страницу приложения и добавляем свой AppleID для тестирования. Возможна дальнейшая публикация в App Store.

Android:
Переместить .apk файл во внутренний накопитель устройства и установить средствами стандартного пакетного менеджера.

Инстуркция по использованию находится в пояснительной записке (глава 6).
